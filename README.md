# game_python
El siguiente repositorio, representa el juego (piedra, papel y tijera) desarrollado en python.
El progrma permite al usuario pasar el rato, disfrutando el clasico juego, con la ventaja que solo necesita de una persona, puesto que va a jugar contra el ordenador.

---

## Para su intalacion debe clonar el repositorio con el siguiente comando: "git clone https://github.com/usuario/game_python.git"

## Ingresar a la carpeta del proyecto
'''
cd game_python
'''

## Ejecutar archivo con la extension .py
'''
python juego.py
'''

---

## Logica del juego
**Reglas utilizadas en el juego son:**
* Piedra gana a tijera.
* Tijera gana a papel.
* Papel gana a piedra.
* Sí ambos son iguales, es un empate.
